Frequently Asked Questions

What types of outlets can you upgrade to?

I install USB-A and USB-C combo outlets, GFCI safety outlets for kitchens and bathrooms, tamper-resistant outlets for child safety, and modern decora rocker-style switches and dimmers. All units fit standard electrical boxes.

How many outlets can you upgrade in one visit?

There is no limit per visit. Booking multiple units at once is more efficient and saves on trip fees. Let me know the quantity when booking.

Do GFCI outlets need to be installed by a specific type of contractor?

No. GFCI outlet installation is standard handyman electrical work that does not require a master electrician license in Texas. I install them correctly to code and test each one before leaving.
