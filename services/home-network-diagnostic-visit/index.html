Frequently Asked Questions

How is this different from the Virtual Starlink Consultation?

The virtual consultation is a video call focused specifically on Starlink systems. This is an in-person visit for any internet or Wi-Fi issue, regardless of provider, where hands-on testing is needed.

If you find I need mesh Wi-Fi or new equipment, is that included?

No — this visit is diagnostic only. If the fix requires installation work like Wi-Fi Mesh & Network Extension, that's booked as a separate visit, and the diagnostic fee may be credited toward it.

What if the problem turns out to be on my ISP's end?

I'll tell you straight if the issue is with your provider rather than your home setup, so you're not paying for work that won't fix it.
