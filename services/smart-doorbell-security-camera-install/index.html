Frequently Asked Questions

Do I need to buy the camera before booking?

Depends on the option you book. If you'd like me to source the device, let me know your property layout and I can advise on a good fit.

Do you run the wiring through the wall or leave it exposed?

I run wiring through the wall or soffit wherever possible for a clean, permanent installation. Surface-run wiring is used only where in-wall routing is not feasible, and it is done neatly with proper weatherproof conduit.

Can you install multiple cameras in one visit?

Yes. Book multiple devices and I will quote accordingly. Multi-camera installs in one visit are more efficient and save on trip fees.
