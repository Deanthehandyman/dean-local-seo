# Dean's Handyman Service — Local SEO Site

Auto-generates and publishes town-specific landing pages.

## One-time setup

### 1. Enable GitHub Pages
Enable GitHub Pages under Settings -> Pages -> Source -> 'GitHub Actions'.

### 2. Add GitHub repo secrets
In Settings -> Secrets and variables -> Actions:
Add GROQ_API_KEY, SITE_URL (deanthehandyman.github.io/dean-local-seo), and INDEXNOW_KEY.

## How to run it manually
Go to the Actions tab, select "Generate Local SEO Pages", and click "Run workflow".
